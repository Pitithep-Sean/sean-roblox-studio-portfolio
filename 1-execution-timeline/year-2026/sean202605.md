### 🧪 [Lab-ID: L260504b] Level 3: Event-Driven Teleportation & Floor Clipping Debugging

- **🧠 Engineering Concept:** การเปิดใช้ระบบดักจับและตอบสนองต่อเหตุการณ์ทางกายภาพ (`.Touched:Connect`) ควบคู่ไปกับการทำความเข้าใจโมเดลโครงสร้างสิ่งมีชีวิตเพื่อตรวจสอบสถานะผู้เล่น (`Humanoid Verification`) ตลอดจนลอจิกป้องกันระบบทำงานซ้ำซ้อน (`Debounce Control`)
    
- 🎬 **Applied Evidence & Frontline Media:**
    
    - 📸 **ภาพสเก็ตช์แผนผังและการแก้บั๊กในหน้ากระดาษของน้องฌอห์น:**
        
    - 🎬 **วิดีโอสาธิตการแก้บั๊กตัวละครตกแมพสำเร็จ:**
        
- 💻 **Luau Source Code:**
```
-- [[Lab-ID: L260504b]]
-- Project: Event-Driven Teleportation (May 2026)
-- Objective: ตรรกะเคลื่อนย้ายตัวละครพร้อมระบบ Debounce เพื่อความเสถียร

local teleportPad = script.Parent
local destination = game.Workspace.DestinationPad

local debounce = false

teleportPad.Touched:Connect(function(hit)
    if not debounce then
        local character = hit.Parent
        local humanoid = character:FindFirstChild("Humanoid")
        
        if humanoid then
            debounce = true -- บล็อคสัญญาณชั่วคราวเพื่อป้องกันสคริปต์ทำงานซ้อนขัดกัน
            local rootPart = character:FindFirstChild("HumanoidRootPart")
            
            if rootPart then
                -- FIX [4May2026]: เพิ่ม Vector3 Offset แกน Y เข้าไป +5 หน่วย เพื่อแก้บั๊กตัวละครจมตกแมพ (Floor Clipping Error)
                rootPart.CFrame = destination.CFrame + Vector3.new(0, 5, 0)
            end
            
            task.wait(2) -- Cooldown Protocol 2 วินาที ก่อนปลดล็อคให้ระบบทำงานต่อได้
            debounce = false
        end
    end
end)
```
