### ## <span style="color:#2e7d32">D260312 : Event-Driven Teleportation System</span>

> <span style="color:#2e7d32">**[Project Progress & Timeline Journey]:**</span>
> ยกระดับจากวัตถุสถิตย์มาเป็นการตอบสนองเชิงพลวัต (Dynamic Interaction) ด้วยการสร้างแผ่นวาร์ปอัจฉริยะ (Teleport Pad) โดยใช้เงื่อนไขคณิตศาสตร์เวกเตอร์ displacement ขยับตำแหน่งตัวละครข้ามแกนพิกัด

> <span style="color:#2e7d32">**[Frontline Behavior & Debugging Resilience]:**</span>
> **เสียงดั้งเดิมของเด็ก:** *"อ้าว! ทำไมตัวละครเหยียบปุ่มวาร์ปแล้วหน้าต่าง Output มันฟ้องตัวแดงว่า HumanoidRootPart is nil อะครับ แล้วตัวละครก็หายไปจากแผนที่เลย!"*
> **[Resilience & Debugging Log]:** บั๊กนี้เกิดจากการที่สคริปต์ตรวจจับพาร์ทแปลกปลอม (เช่น ชิ้นส่วนเสริมหมวกหรือกระเป๋า) ที่วิ่งมาชนก่อนที่ชิ้นส่วนหลักจะมาถึง น้องฌอห์นใช้เวลาร่วม 20 นาทีกางหน้าสคริปต์เพื่อครอบประโยคเงื่อนไข `if` ตรวจเช็คความถูกต้องของออบเจกต์ทีละสเต็ปอย่างมุ่งมั่น

#### 💻 Source Code Verification (ขุมพลังตรรกะหน้างาน)
```lua
-- [[Lab-ID: L260312a]]
local teleportPad = script.Parent
local targetLocation = Vector3.new(0, 50, -25) -- พิกัดปลายทางที่ต้องการให้ตัวละครวาร์ปไป

local function onPadTouched(otherPart)
	-- การใช้ Instance Hierarchy ค้นหาโมเดลของตัวละครผู้เล่น
	local character = otherPart.Parent
	local humanoidRootPart = character:FindFirstChild("HumanoidRootPart")
	
	-- Defensive Programming: ตรวจสอบความปลอดภัยป้องกันสคริปต์เอ๋อ (Fixing Nil Value Bug)
	if humanoidRootPart then
		-- คำนวณ Displacement ด้วย Vector3 Offset เพื่อไม่ให้เท้าผู้เล่นจมลงไปในแผ่นดินปลายทาง
		humanoidRootPart.CFrame = CFrame.new(targetLocation + Vector3.new(0, 3, 0))
	end
end

-- ผูกระบบเข้ากับกลไก Event-Driven Architecture
teleportPad.Touched:Connect(onPadTouched)
``` ^L260312a
