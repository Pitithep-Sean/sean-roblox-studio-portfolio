# TRACK 1 MONTHLY PORTFOLIO DESIGN SPECIFICATION

Version 1.0

## Core Philosophy

Track 1 คือ Learning Journey Timeline

ไม่ใช่เอกสารสอน Roblox

ไม่ใช่ Technical Documentation

ไม่ใช่ Source Code Repository

หน้าที่ของ Track 1 คือแสดงพัฒนาการของผู้เรียนผ่านหลักฐานจริง โดยเชื่อมโยง

- Learning Process
    
- Human Growth
    
- Problem Solving
    
- AI Collaboration
    
- Project-Based Learning
    
- Technical Development
    

เข้าด้วยกันเป็นเรื่องราวเดียว

Track 1 ต้องทำให้กรรมการเข้าใจภายในเวลา 3-5 นาทีว่า

"ผู้เรียนกำลังเติบโตเป็นคนแบบใด"

Technical Deep Dive ให้เชื่อมต่อไปยัง Track 2 ผ่าน Anchor Links และ Tag IDs

# SECTION 3 — Key Learning Milestones

## Purpose

ตอบคำถาม

"อะไรคือช่วงเวลาสำคัญที่สุดของเดือน"

SECTION นี้คือพระเอกของทั้งหน้า

---

## Milestone Types

SECTION นี้รองรับ Milestone ได้ 2 ประเภท

---

### Type A

Technical Learning Milestone

ใช้ Layout

Human Evidence | Technical Evidence

ตัวอย่าง

เด็กกำลังทดลอง Script | Screenshot Lua Script

เด็กกำลัง Debug | Error Screen

เด็กกำลังใช้ AI | Prompt Conversation

เด็กกำลังทดสอบระบบ | System Output

---

หน้าที่

พิสูจน์การเรียนรู้ทางเทคนิค

---

### Type B

Learning Environment Milestone

ใช้ Layout

Photo A | Photo B

ตัวอย่าง

พ่อกำลังสอน Client-Server | เด็กกำลังชี้ Code อธิบาย

Discussion Session | Whiteboard Planning

Mentoring Session | Team Collaboration

---

หน้าที่

พิสูจน์วัฒนธรรมการเรียนรู้

พิสูจน์กระบวนการถ่ายทอดความรู้

พิสูจน์สภาพแวดล้อมการเรียนรู้

ไม่จำเป็นต้องมี Screenshot หรือ Code คู่เสมอไป

---

## Milestone Narrative

Milestone ทุกอันต้องมีคำอธิบาย

ตอบคำถาม

- What happened?
    
- What challenge appeared?
    
- What was learned?
    
- Why does it matter?
    

ความยาว 2-5 บรรทัด
